## Deployment

To deploy this project run

```z-sh
  npm run dev
```

To deploy the laravel 

```z-sh
 php artisan serve
```

## Features

- Light/dark mode toggle
- Color changing on buttons
- Search
- Posting - CRUD
- Like , Comment , Notifications
- User respective profile or wall
- real-time pusher notificatin 
- laravel notify system
- multi_authentication 
- Google , GitHub OAuth
